Projects

Multi-threaded Data Flow System

Based on the Lab Streaming Layer to capturing, filtering, and processing real-time data streams from wireless brainwave devices.

Social Individual Consumption Analysis

Explored social individual consumption patterns by multiple regression models and ensemble methods by converting quarterly-based data to annual data. Cleaning data, categorizing spending data based on family status, and estimating spending based on households. Evaluated model performance using metrics such as R2, MSE, and RMSE, resulting in continuous improvement.

HandWrite Training

Train the Tesseract character library using jTessBoxEditor. By expanding the character library, improve the accuracy of recognition, generate BOX files, and then define character configuration files to achieve character correction.

License Plate Detection and Traffic Statistics

Real-time detection and counting of vehicles for traffic analysis. Using OpenCV for vehicle and license plate detection, extracts features using HOG, classifies the results using an SVM, and tracks vehicles across lanes to generate traffic statistics.
